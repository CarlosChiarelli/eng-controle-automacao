Considerando o terra virtual na entrada inversora ("negativa", tensão zero) do AMPOP, chegamos na relação:

$$ $$$$ \begin{cases} V_{i} = Z_{1}i \\ -V_{o} = Z_{2}i \end{cases} $$$$ $$$$ i = \frac{V_{i}}{Z_{1}}$$$$ $$$$ i = -\frac{V_{0}}{Z_{2}}$$

implicando em

$$ $$$$ G(s) = \frac{V_{o}}{V_{i}} = -\frac{Z_{2}}{Z_{1}}$$

Agora é necessário obter as impedâncias $Z_{1}$ e $Z_{2}$.

$$ $$$$ Z_{1} = R_{1}$$

Para obter $Z_{2}$ precisamos obter $Z_{c}$ (impedância do Capacitor) no domínio de Laplace e obter o equivalente com o resistor 2.

$$ $$$$ i(t) = C\frac{dv}{dx}$$

Aplicando o operador de Laplace em toda equação:

$$ $$$$ I(s) = C[sV - v(0)^{-}]$$$$ $$$$ I(s) = C[sV - 0]$$$$ $$$$ I(s) = sCV$$$$ $$$$ V = \frac{1}{sC}I$$

Logo:

$$ $$$$ V = Z_{c}I$$$$ $$$$ Z_{c} = \frac{1}{sC}$$

Para obter $Z_{2}$ somar as impedância $Z_{c}$ e $R_{2}$ em paralelo:

$$ $$$$ \frac{1}{Z_{2}} = \frac{1}{R_{2}} + \frac{1}{Z_{c}}$$$$ $$$$ \frac{1}{Z_{2}} = \frac{1}{R_{2}} + sC$$$$ $$$$ \frac{1}{Z_{2}} = \frac{1 + sCR_{2}}{R_{2}}$$

Logo:

$$ $$$$ Z_{2} = \frac{R_{2}}{1 + sCR_{2}}$$

Agora basta substituir $Z_{1}$ e $Z_{2}$ na equação de $G(s)$

$$ $$$$ G(s) = -\frac{Z_{2}}{Z_{1}}$$$$ $$$$ G(s) = -\frac{\frac{R_{2}}{1 + sCR_{2}}}{R_{1}}$$$$ $$$$ G(s) = -\frac{R_{2}}{R_{1}}\left(\frac{1}{1 + sCR_{2}}\right)$$

def respostaSistema(f=1, fimTempo=1):
    ''' Calcula a respota do sistema de acordo com a frequencia
    :param f:        Integer é a frequência em Hz
    :param fimTempo: Integer é onde o vetor tempo termina (segundos)
    :return:         Dictionary com 2 Tuples, primeira de (t, V) entrada e segunda de saida (resposta sistema)
    '''
    # intervalo de tempo
    dt = 1e-3
    
    # parametros do filtro
    R1 = 10
    R2 = 10
    C  = 1e-3

    wc = 1  / (R2*C)    # frequencia de corte (rad/s)
    fc = wc / (2*np.pi) # frequencia de corte (Hz)

    # funcao de transferencia
    Num = -R2                                # numerador
    Den = [R1*C*R2, R1]                      # denominador [s^3, s^2, s, 1]
    Gs  = (Num, Den)                         # TF
    t   = np.arange(0, 1, dt)                # tempo (0 a 1s)
    Vi  = np.sin(2*np.pi*f*t)                # tensao de entrada (V)

    t_out, Vo, x_out = lsim(Gs, Vi, t) # simulacao
    
    return {'entrada':(t, Vi), 'saida':(t_out, Vo)}

def respostaSistema2(m=0.1, c=5, k=50):
    ''' Calcula a respota do sistema de acordo com a frequencia
    :param m:        Integer é a massa do sistema
    :param c:        Integer é a constante de amortecimento
    :param k:        Integer é a constante elástica 
    :return:         Dictionary com 3 Tuples, primeira de (t, V) entrada e segunda de saida (resposta sistema), e a terceira propriedades do sistema (m,c,k)
    '''
    # definindo tempo
    dt = 1e-3                # delta
    t  = np.arange(0, 1, dt) # tempo (0 a 1s)

    # funcao de transferencia
    Num = 1                       # numerador
    Den = [m, c, k]               # denominador [s^3, s^2, s, 1]
    Gs  = (Num, Den)              # TF
    
    # entrada
    entrada = {0.1 : 0.01,  # u(0.1) = 0.01  m/s2
               0.6 : 0.005} # u(0.6) = 0.005 m/s2 
    
    u = [entrada[tempo] if tempo in [0.1, 0.6] else 0 for tempo in t]

    t_out, yr, x_out = lsim(Gs, u, t) # simulacao
    
    return {'entrada':(t, u), 'saida':(t_out, yr), 'propriedades':(m,c,k)}

Inicialmente devemos levantar as relações dentro do sistema eletro-mecânico separadamente:

  • Circuito elétrico
$$ $$$$ v(t) = Ri(t) + L\frac{d}{dt}i(t) + e_{a}(t) $$
  • Sistema mecânico
$$ $$$$ T(t) = B\omega(t) + J\dot\omega(t) $$
  • Lei de Faraday de indução
$$ $$$$ e_{a}(t) = k\omega(t) $$
  • Lei da força de Lorentz:
$$ $$$$ T(t) = ki(t) $$

Obtidas todas relações no tempo, vamos manipula-las no domínio de Laplace, aplicando o operador em todas elas:

$$ $$$$ V(s) = [sL + R]I(s) + E_{a}(s) \tag{1}\label{eq:1}$$$$ $$$$ T(s) = [sJ + B]w(s) \tag{2}\label{eq:2}$$$$ $$$$ E_{a} = kw(s) \tag{3}\label{eq:3}$$$$ $$$$ T(s) = kI(s) \tag{4}\label{eq:4}$$

Substitundo (4) em (2) temos:

$$ $$$$ kI = [sJ + B]\omega $$$$ $$$$ I = [sJ + B]\frac{\omega}{k} \tag{5}\label{eq:5}$$

Substitundo (5) e (3) em (1) temos:

$$ $$$$ V = [sL + R]\left([sJ + B]\frac{\omega}{k}\right) + k\omega $$$$ $$$$ V = \left([sL + R][sJ + B]\frac{1}{k}\right)\omega + k\omega $$$$ $$$$ V = \left([sL + R][sJ + B]\frac{1}{k} + k\right)\omega $$$$ $$$$ V = \left(\frac{[sL + R][sJ + B] + k^2}{k} \right)\omega $$

Logo:

$$ $$$$ G(s) = \frac{\omega(s)}{V(s)} $$$$ $$$$ G(s) = \frac{k}{(sL + R)(sJ + B) + k^2} $$
